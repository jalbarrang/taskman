# Plan Ledger Glossary

- **Plan Ledger**: The collection of plans, initiatives, and their execution records.
- **Plan**: A bounded unit of work with tasks and an execution handoff.
- **Task**: An individually trackable item of work within a plan.
- **Initiative**: A grouping of related plans with shared progress and dependencies.
- **Handoff**: The execution document that gives a plan its context and instructions.
- **Projection**: A derived record that summarizes state from the ledger’s source records.
- **Resolution**: Selecting one plan from an explicit name or the active plans in the ledger.
- **Finalizable**: A plan whose task state permits completion.
- **Readiness**: Whether a plan can start because its dependencies are satisfied.
- **Drift**: A disagreement between derived projections and the underlying ledger records.
